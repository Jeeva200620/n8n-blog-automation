# n8n AI Blog Automation Workflow

An advanced, production-ready n8n workflow designed to automate high-quality, SEO-optimized blog article generation and posting. It reads topics from Google Sheets, generates empathetic medical articles using OpenAI / Gemini, fetches relevant stock imagery from Pexels, matches categories, creates tags dynamically, pushes drafts to WordPress, and updates RankMath SEO metadata.

---

## 🌟 Key Features

* **Multi-Language Generation**: Automatically parses language suffixes (e.g. `in Tamil`, `in Hindi`) from Google Sheets titles to write localized content while maintaining short, clean English URLs (slugs).
* **Smart Category Classifier**: Dynamically scans post titles using predefined keyword rules to map posts to target WordPress categories (e.g., *PCOS Treatment*, *Male Infertility Doctor*, *Pregnancy Care*), defaulting safely to *Best Practices And FAQs*.
* **Dynamic Tag Resolver**: Automatically scans tags generated by the AI, queries WordPress to resolve their IDs, and dynamically creates any missing tags before linking them to the post.
* **Empathetic AI Writing style**: Tailored system prompts force the AI to write engaging, simple, medical-grade copy using short paragraphs, clear H2/H3 headings, and structured FAQ sections.
* **Standard WP API Fallback**: Bypasses plugin authorization issues by writing Yoast / RankMath metadata directly into WordPress post meta via standard REST requests.
* **No-Duplicate Marking**: Safely updates rows in Google Sheets without modifying formulas or triggering duplicates.

---

## 🛠️ Workflow Architecture

The workflow executes in a linear, loop-free execution path to ensure absolute reliability and prevent memory leaks:

```
[Start Trigger]
       │
[Get Pending Topics] ──> Reads sheet rows
       │
[Get First Pending] ──> Parses language & sets target topic
       │
[Generate Blog] ──> Employs Chat API for stateless, loop-free writing
       │
[Parse Blog JSON] ──> Robust 3-tier parsing of AI output & category mapping
       │
[Resolve Featured Image] ──> Uploads media, resolves category IDs, and manages tags
       │
[Post to WordPress] ──> Creates a draft post on your WordPress site
       │
[Set SEO Meta] ──> Injects RankMath headers & attaches media/categories/tags
       │
[Mark Done in Sheet] ──> Sets status to 'Done' in Google Sheets
       │
[Done]
```

## 🚀 Setup & Configuration

### 1. Google Sheets Setup
Prepare a spreadsheet with the following header layout:
* **Column A (Title)**: The title of your topic (e.g. `Low AMH Impact on Fertility in Tamil`).
* **Column B (Status)**: Left blank for pending topics. The workflow will update this to `Done` once posted.

### 2. Import into n8n
1. Copy the contents of [ARC_AI_Blog_Automation_FINAL.json](./ARC_AI_Blog_Automation_FINAL.json).
2. Open your n8n workspace, click **Import from File** (or paste it directly onto the canvas using `Ctrl + V`).

### 3. Connect Credentials
Set up the following credentials in your n8n instance:
* **OpenAI / Gemini API** (for article writing)
* **Google Sheets API** (for reading/updating topics)
* **WordPress API** (for post creation)

### 4. Code Nodes Configuration
Double-click the following nodes and update their credential variables:

#### A. **`Resolve Featured Image`**
```javascript
const WP_URL = 'https://your-wordpress-site.com';
const WP_USER = 'YOUR_WP_USERNAME';
const WP_APP_PASSWORD = 'YOUR_WP_APPLICATION_PASSWORD';
```

#### B. **`Set SEO Meta`**
```javascript
const WP_URL = 'https://your-wordpress-site.com';
const WP_USER = 'YOUR_WP_USERNAME';
const WP_APP_PASSWORD = 'YOUR_WP_APPLICATION_PASSWORD';
```
